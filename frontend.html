¿Que es un FrontEnd?

Lado del cliente (HTML, CSS, HTML5, CSS3, Javascript, jQuery)

Los frontends tienden a ser programadores, pero hay diseñadores genios que también hacen frontend. Son los encargados de maquetar la estructura semántica del contenido (HTML), codificar el diseño en hojas de estilo (CSS) y agregar la interacción con el usuario (Javascript).En la época actual los frontends tienen HTML5 y CSS3.
Con HTML5, desde el frontend, es posible hacer geolocalización, dibujo vectorial, guardar datos en el disco del usuario, insertar audio y video, entre otras cosas.Con CSS3, se pueden crear diseños altamente complejos sin la necesidad de imágenes cortadas, sólo usando código. Bordes redondeados, sombras, degradados, fondos múltiples, entre otros.<
Por ultimo, Javascript y sus frameworks añaden el componente de interactividad y conexión al servidor. Es posible comunicarse con el backend y la base de datos sin recargar la página usando AJAX o WebSockets, recibir esos datos y cambiar el diseño entero del sitio. jQuery hace todo esto fácil pero no es el único framework de Javascript

¿Qué hace un front-end developer?

En pocas palabras podemos decir que un front-end developer hace páginas web. Cada vez que navegamos la web en busca de información o de diversión lo hacemos a través de páginas o aplicaciones web que fueron diseñadas por front-end developers (mejor conocidos como diseñadores web). Un front-end developer se dedica a crear páginas e interfaces web atractivas y usables que transmiten información en forma clara y precisa y que, además, permiten a sus usuarios interactuar de diferentes maneras con esa información.
Muchas personas piensan que hacer una página web es una tarea sencilla que consiste en mezclar algunas imágenes, tipografías y colores con la ayuda de editores HTML o de plantillas prediseñadas. Pero en realidad, crear una páginas web desde cero es una tarea muy compleja en la que intervienen muchos factores técnicos y estéticos que son completamente ignorados por la mayoría de la gente.
Para poder crear páginas web un front-end developer requiere de un gran sentido estético y de conocimientos técnicos específicos:

Requiere de un gran sentido estético porque debe lograr que sus diseños sean atractivos y a la vez usables. Por más colorido o bonito que sea un diseño este no sirve de nada si es confuso o si no logra transmitir un mensaje concreto. Un front-end developer tiene que ser capaz de transformar diseños complejos e intrincados en interfaces fáciles de usar y de comprender.
Requiere de conocimientos técnicos específicos porque sin ellos nunca podría lograr que una página o aplicación web funcione correctamente. Hoy en día, entre muchas otras cosas, es necesario que una página o aplicación web tenga un código correctamente estructurado, que sea cargada velozmente, que sea indexada por los buscadores, que esté integrada con las redes sociales, que funcione y se vea bien en todos los navegadores web, que sea compatible con dispositivos móviles como tablets y smartphones y con los televisores más modernos. Para poder hacer todo eso el diseñador web tiene que utilizar y conocer profundamente una gran variedad de lenguajes de programación (ej: HTML, CSS, JavaScript, PHP) y de herramientas de software ( ej:Photoshop, Fireworks).

Además, tiene que estar siempre al tanto de las últimas tendencias sobre diseño y de las últimas innovaciones y avances tecnológicos en materia de desarrollo web porque ninguna otra disciplina relacionada con Internet cambia tan velozmente.
Lo que hace un front-end developer o, lo que es lo mismo, la creación de páginas, sitios o aplicaciones web, lejos de ser una tarea sencilla, es un proceso complejo que plantea cientos de desafíos y que obliga a lidiar con tecnologías y conceptos que evolucionan constantemente.
